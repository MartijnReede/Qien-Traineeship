Bruine tekst

Blauwe boxje --> content

Groene boxje --> padding

oranje boxje --> border

donker oranje --> margin

Als we willen dat de grootte wordt berekend aan de hand van de border (en dus niet inclusief de margin) dan kan je gebruiken in body tag: * { box-sizing: border-box; }

Met overflow-hidden kunnen we content afsnijden die buiten de border valt.
