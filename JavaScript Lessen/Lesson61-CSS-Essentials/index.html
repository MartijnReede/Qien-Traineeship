We kunnen het * teken gebruiken om de stijl aan te passen van alles. Als we daarna specifieke elementen een stijl geven zal dit het * overriden. Als we niet willen dat een bepaalde stijl van een element overridden wordt, kunnen we !important gebruiken. Dat ziet er als volgt uit: body { background-color: red !important; } Met #id kunnen we refereren aan een id, met .class kunnen we refereren aan een class.
